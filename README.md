## 🚀 編譯與執行方式（以 MSYS2 為例）

```bash
g++ main.cpp -o main.exe
./main.exe
```

或使用 Visual Studio Code 內建任務與除錯設定。

## 🔧 開發環境

- 編譯器：GCC 14.2.0 (MSYS2 UCRT64)
- 開發工具：Visual Studio Code
- 作業系統：Windows 10/11
